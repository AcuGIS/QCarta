QCarta
==================================================



QCarta is a QGIS Map Portal and OGC Web Server.

   - Publish QGIS Projects
   - Support for PostGIS
   - Support for most Raster and Vector data files
   - Intergrated MapProxy for Layer Caching
   - Group Level permissions for Stores and Layers
